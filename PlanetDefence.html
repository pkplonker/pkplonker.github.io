Planet Defence - Android Game

My target for the summer following the first year at university was to complete a game that I could release on mobile, specifically to the Android Play Store.

The game was produced over a few weeks and delivers a feature complete game.

Although it still requires clean up,  some polish and more levels. I'm glad to be able to release my first game.

Key functionality implemented:

  • Multiple board sizes, dynamically created at run time.
  • Full menu system, including saved preferences
  • Wave spawning
  • Health, damage, movement and shooting systems
  • CSV import of enemy and wave data to improve development workflow
  • Custom editors to review wave data and player unlocks
  • Autoplay system for debug testing
  • Story
  • Unlock and progression system
